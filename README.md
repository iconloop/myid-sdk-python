## MYID-SDK-PYTHON

`MYID`의 Credential info를 생성/관리/ 검증하기 위한 SDK입니다.

- DID 관련 내용은 [DID Document](#did-document)를 참조 합니다.

### Log
Add the below configuration for loguru.
~~~
LOG_ENABLE_MYID_LOGGER=[true|false]
~~~
